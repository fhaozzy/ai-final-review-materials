# 人工智能期末复习资料库

这个仓库整理了本课程的期中/期末试卷、作业、实验、PPT、AIMA 教材切分页，以及外部备考资料索引。目标是让你或另一个 Codex 复制本目录后，能立刻从同一套材料开始复习、检索考点、补齐错题。

## 先从这里开始

1. 读 [docs/00_开始复习.md](docs/00_开始复习.md)：这是期末复习路线图和第一轮任务。
2. 读 [docs/01_考点题型地图.md](docs/01_考点题型地图.md)：这是根据试卷、作业、实验、PPT 和外部资料综合出的高频题型。
3. 查 [docs/02_资料索引.md](docs/02_资料索引.md)：按主题找到对应 PPT、试卷、作业、实验和教材切片。
4. 从 HMM 往后复习时，读 [docs/06_HMM到后半程复习.md](docs/06_HMM到后半程复习.md)。
5. 给 Codex 接手时，直接把 [AGENTS.md](AGENTS.md) 作为上下文说明。

## 当前复习主线

优先级从高到低：

1. Agent/PEAS、环境类型、智能体结构
2. 搜索：BFS、DFS、UCS、贪婪最佳优先、A*、启发式可采纳性/一致性
3. 局部搜索、博弈搜索、CSP
4. 命题逻辑、一阶逻辑、CNF、归结证明
5. 概率、贝叶斯网络、HMM、朴素贝叶斯
6. 机器学习评估、决策树、感知器/神经网络
7. CNN、卷积、池化、过拟合、ResNet
