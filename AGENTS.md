# Codex 接手说明

这个目录是“人工智能期末复习资料库”，不是普通代码项目。接手时优先完成复习辅助、资料检索、错题整理、考点提炼，而不是重构文件结构。

## 工作目标

- 帮用户按期中/期末试卷、作业、实验、PPT 和教材切片复习。
- 每次回答尽量把题目映射到课程章节、PPT、作业或试卷来源。
- 优先维护 `docs/` 下的复习资料入口和索引。
- 遇到教材/课件/试卷版权问题时，提醒用户区分本地自用、私有仓库和公开 GitHub。

## 高频考点

按考试收益排序：

1. PEAS、Agent 类型、环境属性
2. 搜索算法过程、OPEN 表、路径、完备性、最优性、复杂度
3. A* 的 `f(n)=g(n)+h(n)`，启发式可采纳性与一致性
4. 爬山法、模拟退火、遗传算法
5. minimax 与 alpha-beta 剪枝
6. CSP 三元组、约束图、回溯、约束传播
7. 真值表、CNF、Skolem 化、归结证明
8. 条件概率、Bayes、贝叶斯网络联合分布分解和枚举推理
9. HMM 的过滤、预测、Viterbi 思路
10. 决策树熵和信息增益
11. 训练集/验证集/测试集、K 折、过拟合
12. CNN 结构、卷积核、池化、三通道卷积、ResNet 残差连接

## 编辑约定

- 中文 Markdown 使用 UTF-8。
- 不要删除原始资料；如果要迁移或重命名，先更新 `docs/02_资料索引.md`。
- 不要把根目录 AIMA 原书 PDF、课程压缩包或超过 100MB 的文件加入普通 Git 历史。
- 公开仓库只提交你有权公开的资料；本地自用索引可以引用未上传的私有资料路径。
