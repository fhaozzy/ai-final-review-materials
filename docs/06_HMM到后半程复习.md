## 1. 马尔可夫链

先掌握三个对象：

| 对象 | 含义 | 考试写法 |
| --- | --- | --- |
| 状态 `X_t` | 第 `t` 个时间片的真实状态 | 例如今天是否下雨、网页所在位置 |
| 转移模型 | 当前状态如何影响下一状态 | `P(X_t | X_{t-1})` |
| 初始分布 | 起始时刻各状态概率 | `P(X_0)` 或 `P(X_1)` |

核心假设是一阶马尔可夫性：

```text
P(X_t | X_0, ..., X_{t-1}) = P(X_t | X_{t-1})
```

联合分布可以写成：

```text
P(X_0, X_1, ..., X_t) = P(X_0) * Π P(X_i | X_{i-1})
```

要会做的题：

- 给转移矩阵和初始分布，算若干步后的状态分布。
- 知道稳定分布/平稳分布是满足 `π = πT` 的分布。
- PageRank 可以理解为马尔可夫链稳定分布的应用。

## 2. HMM

HMM 比马尔可夫链多了一个“看得见的证据层”。

| 对象 | 含义 | 常用符号 |
| --- | --- | --- |
| 隐状态 | 真正变化但不能直接观察的状态 | `X_t` |
| 观测/证据 | 每个时刻看到的现象 | `E_t` |
| 转移模型 | 隐状态之间如何转移 | `P(X_t | X_{t-1})` |
| 传感器模型/观察模型 | 隐状态如何产生观测 | `P(E_t | X_t)` |

两个关键独立性假设：

```text
P(X_t | X_0, ..., X_{t-1}) = P(X_t | X_{t-1})
P(E_t | X_0, ..., X_t, E_0, ..., E_{t-1}) = P(E_t | X_t)
```

HMM 联合分布模板：

```text
P(X_0:t, E_1:t) = P(X_0) * Π P(X_i | X_{i-1}) * Π P(E_i | X_i)
```

考试最常见不是让你推整套算法，而是让你识别：

- 哪些是隐状态，哪些是观测。
- 转移概率表和传感器概率表分别是什么。
- 某一时刻的观测如何更新对隐状态的信念。

如果出现计算题，优先使用过滤的两步模板：

```text
预测：prior_t = Σ_x P(X_t | x) * belief_{t-1}(x)
更新：belief_t ∝ P(e_t | X_t) * prior_t
```

写答案时记得最后归一化。

## 3. 机器学习评估

2024 期末回忆明确考了这一块。

| 概念 | 一句话定义 | 作用 |
| --- | --- | --- |
| 训练集 | 用来拟合模型参数的数据 | 学模型 |
| 验证集 | 用来调参、选模型的数据 | 防止只对测试集调参 |
| 测试集 | 最后评估泛化能力的数据 | 模拟未知数据 |
| K 折交叉验证 | 数据分成 `K` 份，轮流用 1 份测试、其余训练 | 小数据集上更稳定地估计性能 |

监督/无监督/半监督：

- 监督学习：训练数据有标签，适合分类、回归。
- 无监督学习：训练数据无标签，适合聚类、降维、发现结构。
- 半监督学习：少量有标签 + 大量无标签，适合标注成本高的任务。

过拟合模板：

```text
过拟合：训练集表现好，测试集表现差。
原因：模型太复杂，记住训练样本细节或噪声。
防治：更多数据、正则化、早停、交叉验证、Dropout、数据增强、降低模型复杂度。
```

## 4. 朴素贝叶斯

核心是假设特征在类别给定时条件独立：

```text
P(x_1, ..., x_n | c) = Π P(x_i | c)
```

分类模板：

```text
选择使 P(c) * Π P(x_i | c) 最大的类别 c
```

要会解释：

- `P(c)` 是先验概率。
- `P(x_i | c)` 是类别 `c` 下第 `i` 个特征出现的条件概率。
- 拉普拉斯平滑用于避免某个条件概率为 0 导致整项乘积为 0。

## 5. 决策树

2024 期末回忆里决策树占 20 分，要重点练计算。

信息熵：

```text
H(S) = -Σ p_i log2(p_i)
```

按属性 `A` 划分后的条件熵：

```text
H(S | A) = Σ (|S_v| / |S|) * H(S_v)
```

信息增益：

```text
Gain(S, A) = H(S) - H(S | A)
```

做题步骤：

1. 先数总样本里正负类个数，算 `H(S)`。
2. 对每个候选划分分别数子集正负类，算 `H(S_v)`。
3. 加权求 `H(S | A)`。
4. 算 `Gain(S, A)`，选信息增益最大的属性。
5. 子集纯了就停止；不纯就继续递归。

## 6. 神经网络、CNN、ResNet

2024 期末回忆里 CNN 占 20 分，不要只背“深度学习很强”。

CNN 结构模板：

```text
输入图像 -> 卷积层 -> 激活函数 -> 池化层 -> 若干卷积/池化 -> 展平 -> 全连接层 -> Softmax/分类输出
```

卷积层作用：

- 提取局部特征，例如边缘、纹理、局部形状。
- 多个卷积核会产生多个特征图。
- 三通道卷积时，每个卷积核也有三个通道，对应输入的三个通道，逐通道卷积后求和。

池化层作用：

- 降低特征图尺寸。
- 减少计算量。
- 提高局部平移鲁棒性。

CNN 防过拟合答题模板，任选两到四点：

- 数据增强。
- Dropout。
- 正则化。
- 早停。
- 增加训练数据。
- 降低模型复杂度。
- 使用验证集/K 折交叉验证调参。

ResNet 核心思想：

```text
残差连接让网络学习 F(x) = H(x) - x，输出写成 H(x) = F(x) + x。
```

它的作用是缓解深层网络训练困难、梯度消失和退化问题，让更深的网络更容易优化。
