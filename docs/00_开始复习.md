# 00 开始复习

这门课最稳的复习方法不是平均看书，而是用试卷和作业反推教材主线。当前资料显示，考试主轴是：

```text
Agent/PEAS -> 搜索 -> 局部搜索/博弈/CSP -> 逻辑 -> 概率/BN/HMM -> 机器学习 -> CNN/ResNet
```

## 今天就做这 4 件事

1. 用 30 分钟读完 `学习规划.md` 和本文件，先把课程地图搭起来。
2. 用 60 分钟复习 Agent/PEAS：看 `ppt/第二讲.pptx`，再做 `作业/第一次作业/作业答案.md` 的第 1-2 题。
3. 用 90 分钟复习搜索：看 `ppt/第三讲.pptx`、`ppt/第四讲.pptx`，再做作业 1 第 3-5 题。
4. 最后用 30 分钟打开 `专业核心-人工智能（93）/2024期末考试.md`，只做题型识别：每道题归到哪一章、用哪种标准解法。

## 三轮复习法

第一轮：看 PPT 和教材切片建骨架。  
目标是能说清每章解决什么问题、核心对象是什么、常考公式/算法是什么。

第二轮：用作业和实验把方法练熟。  
每学完一个主题，立刻回到对应作业题和实验报告，不要只停留在概念。

第三轮：刷试卷和错题。  
重点不是多刷，而是把每道题归类到固定模板：PEAS、搜索表、归结证明、BN 分解、熵和信息增益、CNN 结构题。

## 最小闭环

如果时间很少，先完成这个闭环：

1. Agent：会写 PEAS、环境属性、Agent 类型。
2. 搜索：会画搜索树，会算 BFS/DFS/UCS/贪婪/A* 的展开顺序。
3. 逻辑：会把自然语言翻成一阶逻辑，并用归结证明。
4. BN：会写联合分布分解，能代入 CPT 计算后验概率。
5. 决策树：会算熵、条件熵、信息增益，并能选根节点。
6. CNN：会解释结构，能识别卷积核大小、步长、池化，知道如何防过拟合。
