## 1. 第 14 讲：深度学习基础

神经网络可以理解为多层函数复合：

```text
输入 x -> 线性变换 z = wx + b -> 激活函数 a = f(z) -> 输出 y
```

必须会解释：

- 线性回归输出连续值，逻辑回归输出分类概率。
- 激活函数让网络具备非线性表达能力，否则多层线性层仍然等价于一层线性层。
- 损失函数衡量预测值和真实值之间的差距。
- 梯度下降沿着损失函数下降最快的方向更新参数。
- 反向传播用链式法则从输出层往前计算每个参数对损失的影响。

考试答题模板：

```text
前向传播计算预测值；损失函数衡量预测误差；反向传播利用链式法则计算梯度；梯度下降根据梯度更新参数，使损失逐步降低。
```

## 2. 第 15 讲：CNN

CNN 的三个动机：

- 局部感受野：图像中的局部模式通常只占小区域，不必每个神经元连接整张图。
- 参数共享：同一种模式可能出现在图像不同位置，可以用同一个卷积核检测。
- 下采样/池化：降低尺寸和计算量，并增强局部平移鲁棒性。

卷积输出尺寸：

```text
W2 = (W1 - F + 2P) / S + 1
H2 = (H1 - F + 2P) / S + 1
D2 = K
```

其中：

- `W1, H1, D1` 是输入宽、高、通道数。
- `F` 是卷积核大小。
- `S` 是步长。
- `P` 是 zero-padding 大小。
- `K` 是卷积核数量，也就是输出通道数。

卷积层参数量：

```text
参数量 = (F * F * D1) * K + K
```

其中最后的 `+K` 是每个卷积核一个偏置。

池化层作用：

- 降低特征图尺寸。
- 减少计算量和参数风险。
- 提高局部平移鲁棒性。

防过拟合常用答案：

- 数据增强。
- Dropout。
- 正则化。
- 早停。
- 增加训练数据。
- 使用全局池化降低参数量。

## 3. 第 16 讲：RNN、LSTM、Transformer

RNN 适合序列数据，因为它有隐藏状态：

```text
h_t = f(x_t, h_{t-1})
```

常见输入输出模式：

| 模式 | 例子 |
| --- | --- |
| many-to-one | 情感分类、视频分类 |
| one-to-many | 图像字幕生成、音乐生成 |
| many-to-many | 序列标注 |
| sequence-to-sequence | 机器翻译、摘要生成 |

普通 RNN 的问题：

```text
长序列反向传播时，梯度反复乘矩阵，容易梯度消失或梯度爆炸，导致远距离信息难以保留。
```

LSTM/GRU 的核心：

```text
用门控结构决定哪些信息保留、遗忘或输出，从而更好地建模长程依赖。
```

Transformer 的核心：

- 自注意力让每个词直接关注序列中其他词。
- 多头注意力从多个子空间学习不同关系。
- 位置编码补充序列顺序，因为自注意力本身不天然知道词的位置。

考试答题模板：

```text
RNN 通过隐藏状态传递历史信息，适合序列建模；LSTM/GRU 通过门控结构缓解长程依赖和梯度消失；Transformer 用自注意力并行建模序列中任意位置之间的关系，并用位置编码表示顺序。
```

## 4. 第 17 讲：强化学习基础

强化学习研究的是序列决策：

```text
Agent 在环境中观察状态 s，选择动作 a，得到奖励 r 和下一状态 s'，目标是最大化长期累积奖励。
```

核心术语：

| 术语 | 含义 |
| --- | --- |
| Agent | 做决策的智能体 |
| Environment | 与智能体交互的环境 |
| State `s` | 当前局面 |
| Action `a` | 智能体可执行的动作 |
| Reward `r` | 环境给出的即时反馈 |
| Policy `π(a|s)` | 在状态 `s` 下选择动作 `a` 的概率 |
| Transition `p(s'|s,a)` | 执行动作后转移到下一状态的概率 |
| Trajectory | `s1,a1,r1,s2,a2,r2,...` 的交互序列 |
| Episode | 从开始到终止的一次完整交互 |

折扣回报：

```text
U_t = R_t + γR_{t+1} + γ^2R_{t+2} + ...
```

其中 `γ` 是折扣因子，越接近 1 越重视长期奖励，越接近 0 越重视眼前奖励。

价值函数：

```text
Vπ(s)：在策略 π 下，从状态 s 出发的期望回报。
Qπ(s,a)：在策略 π 下，从状态 s 先执行动作 a 后的期望回报。
```

## 5. 第 18 讲：DQN、TD 与策略梯度

如果知道最优动作价值函数：

```text
Q*(s,a)
```

那么最优动作就是：

```text
a* = argmax_a Q*(s,a)
```

DQN 的思想：

```text
用神经网络 Q(s,a;w) 近似动作价值函数 Q(s,a)。
```

TD 学习的关键恒等式：

```text
U_t = R_t + γU_{t+1}
```

DQN 常用 TD 目标：

```text
target = r + γ max_a Q(s', a; w)
```

训练时让当前估计接近 TD 目标：

```text
Q(s,a;w) -> r + γ max_a Q(s',a;w)
```

策略梯度的核心思想：

```text
不只学习价值函数，而是直接学习策略 π(a|s;θ)，让高回报动作出现概率变大。
```

## 6. 最小自测

1. 为什么神经网络需要非线性激活函数？
2. 一个输入 `32 x 32 x 3`，卷积核 `F=3`，步长 `S=1`，填充 `P=1`，卷积核数量 `K=64`，输出尺寸是多少？
3. 上一题卷积层参数量是多少？
4. RNN 为什么比普通前馈网络更适合文本或时间序列？
5. LSTM/GRU 主要解决普通 RNN 的什么问题？
6. Transformer 为什么需要位置编码？
7. 强化学习里 `Vπ(s)` 和 `Qπ(s,a)` 有什么区别？
8. DQN 的 TD target 怎么写？

答案：

1. 非线性激活让多层网络能表达非线性函数，否则多层线性变换仍等价于一层线性变换。
2. 输出尺寸是 `32 x 32 x 64`。
3. 参数量是 `(3 * 3 * 3) * 64 + 64 = 1792`。
4. RNN 有隐藏状态 `h_t`，能把之前时刻的信息传到当前时刻。
5. 主要缓解长程依赖、梯度消失和信息遗忘问题。
6. 自注意力本身不含顺序信息，位置编码用于告诉模型每个词在序列中的位置。
7. `Vπ(s)` 评价状态，`Qπ(s,a)` 评价状态下先采取某个动作的好坏。
8. `target = r + γ max_a Q(s', a)`。
