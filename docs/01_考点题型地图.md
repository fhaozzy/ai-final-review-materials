# 01 考点题型地图

本文件综合了本地期中/期末资料、作业答案、实验报告、PPT、AIMA 教材切片，以及 `docs/03_外部资料_CSDN.md` 中的外部备考清单。

## 题型 1：Agent 与 PEAS

常见问法：

- 给一个系统，写 PEAS。
- 判断环境属性：完全/部分可观测、确定/随机、静态/动态、离散/连续、单/多智能体。
- 选择合适的 Agent 类型：简单反射、基于模型、基于目标、基于效用。

## 题型 2：搜索算法过程与性质

常见问法：

- 画搜索树，写 DFS/BFS/UCS/贪婪/A* 的扩展顺序。
- 给 `g(n)`、`h(n)`，判断下一步展开哪个结点。
- 比较完备性、最优性、时间复杂度、空间复杂度。
- 证明一致性推出可采纳性。

## 题型 3：局部搜索、博弈搜索、CSP

常见问法：

- 比较爬山法和模拟退火，说明模拟退火是否保证最优。
- 写遗传算法流程：编码、适应度、选择、交叉、变异。
- minimax/alpha-beta 剪枝过程。
- 将数独、八皇后、地图着色等问题形式化为 CSP。

## 题型 4：逻辑、CNF 与归结证明

常见问法：

- 用真值表判断公式是否有效、可满足、不可满足。
- 将公式转成 CNF。
- 把自然语言写成一阶逻辑。
- 用归结反证法证明结论。

2024 期末回忆里出现了“先写出，再推理”的自然语言证明题，所以这一块不能只背定义，要练完整书写。

## 题型 5：概率、贝叶斯网络、HMM

常见问法：

- 条件概率和 Bayes 公式计算。
- 判断变量独立/条件独立。
- 根据贝叶斯网络写联合分布分解。
- 用枚举法计算后验概率。
- HMM 的转移模型、传感器模型、过滤、预测、最可能序列。

## 题型 8：神经网络、CNN、ResNet

常见问法：

- 简述 CNN 结构：卷积层、激活层、池化层、全连接层、输出层。
- 看图判断卷积核尺寸、卷积核个数、步长、池化步长。
- 计算三通道卷积。
- 写 CNN 防止过拟合的方法。
- 解释 ResNet 的残差连接为什么能帮助深层网络训练。
