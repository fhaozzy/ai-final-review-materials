## 1. 先分清对象

HMM 是“隐藏状态随时间变化，但每个时刻只能看到一个带噪声的观测”的模型。

| 名称 | 符号 | 含义 | 例子 |
| --- | --- | --- | --- |
| 隐状态 | `X_t` | 第 `t` 时刻真实但不能直接观察的状态 | 是否下雨、骰子类型、机器人位置 |
| 观测/证据 | `E_t` | 第 `t` 时刻能看到的现象 | 是否带伞、掷出的点数、传感器读数 |
| 初始分布 | `P(X_0)` | 起点时各隐状态的概率 | `P(Rain_0)=0.5` |
| 转移模型 | `P(X_t | X_{t-1})` | 上一时刻隐状态如何影响当前隐状态 | 今天下雨会影响明天下雨概率 |
| 传感器模型 | `P(E_t | X_t)` | 当前隐状态如何产生观测 | 下雨时看到伞的概率较高 |

考试写法：看到题目先画两层链。

```text
X_0 -> X_1 -> X_2 -> ... -> X_t
       |      |             |
       E_1    E_2           E_t
```

## 2. 三个必须会默写的公式

一阶马尔可夫假设：

```text
P(X_t | X_0, ..., X_{t-1}) = P(X_t | X_{t-1})
```

观测独立假设：

```text
P(E_t | X_0, ..., X_t, E_1, ..., E_{t-1}) = P(E_t | X_t)
```

HMM 联合分布：

```text
P(X_0:t, E_1:t) = P(X_0) * Π_{i=1..t} P(X_i | X_{i-1}) * Π_{i=1..t} P(E_i | X_i)
```

## 3. 过滤和预测的计算模板

过滤要算的是：

```text
P(X_t | e_1, ..., e_t)
```

意思是：已经看到从 1 到 `t` 的证据后，当前隐藏状态的概率分布。

标准两步：

```text
预测：prior_t(x') = Σ_x P(x' | x) * belief_{t-1}(x)
更新：belief_t(x') = α * P(e_t | x') * prior_t(x')
```

其中：

- `belief_{t-1}(x)` 是上一时刻过滤后的信念分布。
- `prior_t(x')` 是还没看第 `t` 个观测时，对当前状态的预测。
- `P(e_t | x')` 是传感器模型。
- `α` 是归一化常数，让所有概率加起来等于 1。

预测要算的是：

```text
P(X_{t+k} | e_1, ..., e_t)
```

意思是：只根据已有证据，推未来状态。做法是从当前 `belief_t` 出发，反复乘转移模型，不再乘新的观测概率。

```text
belief_{t+1|t}(x') = Σ_x P(x' | x) * belief_t(x)
```

## 4. Viterbi 最可能序列

Viterbi 要算的不是“某个时刻哪个状态概率最大”，而是“整条隐藏状态序列最可能是哪一条”：

```text
argmax_{x_1:t} P(x_1:t | e_1:t)
```

答题思路：

1. 每个时刻、每个状态保存一条到这里为止的最优路径概率。
2. 从上一时刻所有状态转移过来，取概率最大的那条。
3. 再乘当前观测概率。
4. 保存回溯指针，最后从最大终点倒推整条路径。

递推模板：

```text
m_t(x') = P(e_t | x') * max_x [P(x' | x) * m_{t-1}(x)]
```

## 5. 常见题型答题模板

### 题型 A：指出 HMM 的组成

答题顺序：

1. 隐状态是 `X_t = ...`。
2. 观测是 `E_t = ...`。
3. 转移模型是 `P(X_t | X_{t-1})`，描述状态随时间如何变化。
4. 传感器模型是 `P(E_t | X_t)`，描述状态如何产生观测。
5. 初始分布是 `P(X_0)` 或 `P(X_1)`。

### 题型 B：写联合分布分解

直接套：

```text
P(x_0, x_1, ..., x_t, e_1, ..., e_t)
= P(x_0) * P(x_1|x_0) * ... * P(x_t|x_{t-1})
  * P(e_1|x_1) * ... * P(e_t|x_t)
```

### 题型 C：做一轮过滤计算

答题顺序：

1. 用上一时刻 `belief` 和转移矩阵先算 `prior`。
2. 用当前观测的传感器概率乘 `prior`。
3. 把两个或多个状态的未归一化概率相加。
4. 每一项除以总和，得到归一化后的 `belief`。

## 6. 10 分钟自测题

给定 Weather HMM：

```text
P(Rain_0)=0.5, P(NotRain_0)=0.5

P(Rain_t | Rain_{t-1}) = 0.7
P(NotRain_t | Rain_{t-1}) = 0.3
P(Rain_t | NotRain_{t-1}) = 0.3
P(NotRain_t | NotRain_{t-1}) = 0.7

P(Umbrella_t | Rain_t) = 0.9
P(Umbrella_t | NotRain_t) = 0.2
```

问题：

1. 这个 HMM 的隐状态和观测分别是什么？
2. 写出 `P(Rain_0, Rain_1, Umbrella_1)` 的分解式。
3. 看到第 1 天有伞 `Umbrella_1` 后，计算 `P(Rain_1 | Umbrella_1)`。
4. 如果第 2 天还没看到新观测，只做预测，计算 `P(Rain_2 | Umbrella_1)`。

答案：

第 1 题：

```text
隐状态：是否下雨，即 Rain_t / NotRain_t
观测：是否看到伞，即 Umbrella_t / NotUmbrella_t
```

第 2 题：

```text
P(Rain_0, Rain_1, Umbrella_1)
= P(Rain_0) * P(Rain_1 | Rain_0) * P(Umbrella_1 | Rain_1)
```

第 3 题：

先预测：

```text
prior(Rain_1) = 0.7 * 0.5 + 0.3 * 0.5 = 0.5
prior(NotRain_1) = 0.3 * 0.5 + 0.7 * 0.5 = 0.5
```

再更新：

```text
未归一化 Rain_1 = 0.9 * 0.5 = 0.45
未归一化 NotRain_1 = 0.2 * 0.5 = 0.10
总和 = 0.55

P(Rain_1 | Umbrella_1) = 0.45 / 0.55 = 0.818
P(NotRain_1 | Umbrella_1) = 0.10 / 0.55 = 0.182
```

第 4 题：

```text
P(Rain_2 | Umbrella_1)
= 0.7 * 0.818 + 0.3 * 0.182
= 0.627
```
